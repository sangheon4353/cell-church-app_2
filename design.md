# 셀 모임 앱 디자인 계획

## 브랜드 컨셉
- **앱 이름**: 셀모임 (CellMeet)
- **슬로건**: 함께 자라는 신앙
- **분위기**: 따뜻하고 차분한 영적 공동체 느낌

## 화면 목록

### 인증 플로우
1. **온보딩 화면** - 앱 소개 슬라이드 (3장)
2. **로그인 화면** - 전화번호 + 비밀번호
3. **회원가입 화면** - 이름, 성별, 생년월일, 전화번호, 셀 리더 선택
4. **승인 대기 화면** - 셀 리더 승인 대기 안내

### 메인 탭 (승인 후)
5. **홈 화면** - 오늘 날짜, 매일성경 범위, 성경 쓰기 현황, 기도 시간 현황
6. **성경 쓰기 화면** - 셀원 진행률 리스트
7. **성경 입력 화면** - 책/장/절 선택 입력
8. **기도 시간 화면** - 기도 시간 입력 및 현황
9. **마이페이지 화면** - 개인 정보, 설정

### 셀 리더 전용
10. **리더 관리 화면** - 가입 승인/거절, 성경 권 할당, 셀원 현황

## 화면별 상세 설계

### 1. 온보딩 화면
- 3개 슬라이드: 앱 소개 → 성경 쓰기 → 기도 시간
- 각 슬라이드: 일러스트 아이콘 + 제목 + 설명
- 하단: 페이지 인디케이터 + "시작하기" 버튼
- 건너뛰기 버튼 (우측 상단)

### 2. 로그인 화면
- 상단: 앱 로고 + 이름
- 전화번호 입력 필드
- 비밀번호 입력 필드
- 로그인 버튼 (primary)
- "회원가입" 링크

### 3. 회원가입 화면
- 스텝 형식 (1/3 → 2/3 → 3/3)
- Step 1: 이름, 성별 (남/여 토글)
- Step 2: 생년월일 (날짜 피커), 전화번호 + 인증 버튼
- Step 3: 셀 리더 선택 (드롭다운 목록)
- 진행률 바 상단 표시

### 4. 승인 대기 화면
- 중앙 일러스트 (대기 아이콘)
- "승인 대기 중입니다" 메시지
- 셀 리더 이름 표시
- 새로고침 버튼

### 5. 홈 화면
- **상단 헤더**: 날짜, 사용자 이름 인사
- **매일성경 카드**: 오늘의 성경 범위 + 큐티 바로가기 버튼
- **성경 쓰기 카드**: 이번 주 총 절 수 + 진행률 바 + 상세 보기 버튼
- **기도 시간 카드**: 이번 주 합산 시간 / 목표 + 진행률 바 + 입력 버튼
- 카드형 UI, 둥근 모서리 (16px), 부드러운 그림자

### 6. 성경 쓰기 상세 화면
- 셀원 리스트 (FlatList)
- 각 셀원 카드: 이름, 할당된 성경 권, 진행률 바, 완료 장수
- 내 카드 강조 표시
- 상단: 셀 전체 총 절 수 요약

### 7. 성경 입력 화면 (모달/시트)
- 책 선택 (할당된 책만, 드롭다운)
- 장 선택 (숫자 스크롤)
- 절 범위 선택 (시작 절 ~ 끝 절)
- 완료 버튼
- 이미 입력된 절 표시 (비활성화)

### 8. 기도 시간 화면
- 이번 주 내 기도 시간 합계
- 빠른 입력 버튼: +1분, +5분, +10분
- 직접 입력 필드
- 이번 주 셀 전체 합산 진행률

### 9. 마이페이지
- 프로필 카드 (이름, 셀, 리더)
- 내 성경 쓰기 통계
- 내 기도 시간 통계
- 설정 (알림, 다크모드)
- 로그아웃

### 10. 리더 관리 화면 (셀 리더 전용)
- 탭: 승인 대기 | 셀원 목록
- 승인 대기 탭: 대기자 카드 + 승인/거절 버튼
- 셀원 목록 탭: 셀원별 성경 권 할당 버튼, 활동 현황

## 핵심 사용자 플로우

### 신규 사용자 플로우
온보딩 → 회원가입 (3단계) → 승인 대기 → (리더 승인) → 홈

### 성경 쓰기 입력 플로우
홈 → 성경 쓰기 카드 탭 → 상세 화면 → 입력 버튼 → 책/장/절 선택 → 완료

### 기도 시간 입력 플로우
홈 → 기도 시간 카드 → +N분 버튼 탭 → 즉시 반영

### 셀 리더 승인 플로우
리더 관리 탭 → 승인 대기 목록 → 승인/거절 → 성경 권 할당

## 컴포넌트 디자인 원칙
- 카드: `rounded-2xl`, `shadow-sm`, `border border-border`, `bg-surface`
- 버튼 (primary): `bg-primary`, `rounded-xl`, `py-3`, `px-6`
- 버튼 (secondary): `border border-primary`, `rounded-xl`
- 진행률 바: 높이 8px, 둥근 끝, 배경 `bg-border`, 채움 `bg-primary`
- 입력 필드: `bg-surface`, `border border-border`, `rounded-xl`, `p-4`
- 여백: 화면 패딩 `p-5`, 카드 내부 `p-4`, 요소 간격 `gap-3`
